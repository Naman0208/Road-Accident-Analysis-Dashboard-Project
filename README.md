# 🚗 Road Accident Analysis (UK) | Power BI

## 🚀 Project Overview
This project analyzes **road accident data in the UK** using an interactive Power BI dashboard.

The dataset contains accident and casualty data for **two years**, allowing year-over-year comparison and trend analysis.

The goal is to identify:
- Accident patterns  
- Casualty trends  
- High-risk conditions  
- Key factors affecting road safety  

---

## 📊 Dashboard Highlights

The dashboard provides a comprehensive overview of road accidents using **DAX-powered KPIs and interactive visuals**.

### 🔹 Key KPIs (Using DAX)

- **Current Year (CY) Casualties**
- **Current Year (CY) Accidents**
- **Fatal Casualties**
- **Serious Casualties**
- **Slight Casualties**
- **Year-over-Year (YoY) Comparison**

---

## 📈 Key Visualizations

- 📅 **Monthly Trend Analysis** (CY vs Previous Year)  
- 🚘 **Casualties by Vehicle Type** (Car, Bike, Bus, Van, etc.)  
- 🌆 **Urban vs Rural Casualties Distribution**  
- 🌙 **Daylight vs Darkness Impact**  
- 🛣️ **Casualties by Road Type**  
- 🌍 **Geographical Map of Accidents**  
- 🌦️ **Weather & Road Condition Filters**  

---

## 📌 Key Insights

- Majority of accidents occur in **urban areas**  
- **Cars** contribute to the highest number of casualties  
- Most accidents happen during **daylight conditions**  
- **Single carriageways** have the highest accident rates  
- Overall casualties show a **decline compared to the previous year**  

---

## 💡 Business / Safety Recommendations

- Improve safety measures in **urban areas**  
- Focus on **high-risk road types** like single carriageways  
- Enhance **driver awareness programs**  
- Strengthen safety regulations for high-risk vehicle categories  
- Use targeted interventions based on weather and road conditions
